Introduction Elements

Introduction

A simple pure css, pure js responsive theme.

This theme is not designed for the mainstream use but for personal project use. Only a subset of html was themed in the css file. Images are not recommended as they are note themed neither are videos. This stuff doesn't match the theme for the old times.

Elements

The following elemetns are supported by web95 theme.

NavBar

Navigation bars are supported by default and as you can probabbly see a navigation bar (grey colored strip) is already their at this page. 

<div class="navbar">
  <a class="navbar-button" href="#"> Home </a>
  <a class="navbar-button" href="#about-us"> About </a>
</div>

will result in

Home About

Headings

Headings have been designed with size in mind, all headings by default will have ~ around them. 

<h1> This is Heading 1 </h1>
<h2> This is Heading 2 </h2>
<h3> This is Heading 3 </h3>
<h4> This is Heading 4 </h4>
<h5> This is Heading 5 </h5>
<h6> This is Heading 6 </h6>

will result in

This is Heading 1

This is Heading 2

This is Heading 3

This is Heading 4

This is Heading 5
This is Heading 6

Button

Buttons are flat for this theme 

<div class="button"> My Button </div>
<div class="button"> My Button 2 </div> <div class="button"> My Button 3 </div>

will result in

My Button
My Button 2
My Button 3
